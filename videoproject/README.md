
### 项目名称

基于django的视频点播网站开发

### 项目功能
本项目分为前台和后台

前台功能
- 视频列表展示
- 视频播放详情
- 详情评论
- 个人中心

后台功能
- 视频管理
- 评论管理
- 用户管理
- 反馈管理



### 适合人群

python初级学员、大学生、系统设计人员

### 技术栈
python/django/nginx/mysql/semantic-css/jquery/html
